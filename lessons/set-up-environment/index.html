Github (git repository server)

После того как мы скачали все необходимые программы и сделали акаунт в Github, нам нужно синхронизовать Github с местным репозиторием. Для этого заходим на Github под свой акаунт. Создаём проект test нажимаем Create Repository. Копируем ссылку, идем на наш локальный сервер xampp\htdocs нажимаем правой кнопкой мыши и выбираем Git Clone...TortoiseGit предлогает дать имя Directory называем test и жмем ок. После создаем проект в NetBeans, выбираем HTML5/JavaScript-HTML5/JS Aplication жмем next называем NetBeansTest бует находится в xampp\htdocs жмем next-next-finish
