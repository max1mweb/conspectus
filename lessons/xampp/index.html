Что такое XAMPP и для чего он нужен?

XAMPP - сборка Web-сервера, в которой содержится набор приложений для полноценного функционирования сайта. При этом не требуется настройка каждого компонента по отдельности. Сборка позволяет вести разработку на отдельном компьютере без подключения к сети. Соответственно, нет необходимости в высокоскоростном доступе в Интернет и наличии хостинга.

Проверка работоспособности XAMPP

Для этого через NetBeans в директории C:\Xampp\htdocs создаем папку lessons. В Lessons создаем файл index.html В index.html прописываем Hello World! Открываем панель управления XAMPP. Запускаем Apache кнопкой start. Теперь в любом браузере пройдите по адресу: http://localhost/lessons и, если всё сделано правильно, вы увидите надпись «Hello World!»
